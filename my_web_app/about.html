About the Application

This application is designed for central bank supervisors and authorized personnel from private financial institutions to evaluate and monitor cybersecurity maturity across institutions using a structured, control-based framework.

Key Features

  • Customizable library of cybersecurity controls
  • Group-based control visibility for different institution sizes
  • Self-assessment capabilities for bank users
  • Supervisor evaluation and observation periods
  • Multilingual support across 10 languages
  • Comprehensive reporting tools
